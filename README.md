# 社区系统前端项目

---

后端仓库：**[community](https://github.com/ibaiq/community)**

## Vue 3 + Vite

### 使用项目

``` shell
# 克隆仓库
git clone https://github.com/ibaiq/community-ui-admin-arco.git
# 或者
git clone git@github.com:ibaiq/community-ui-admin-arco.git

cd community-ui-admin-arco

npm i
# 或者
npm install

# 开发环境
npm run dev

# 构建打包
npm run build
```
